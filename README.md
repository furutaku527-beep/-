# 日本株デイトレード手法 バックテストシステム

YouTuber「シンジ」氏の手法(noteで公開)が、実際に売買して利益が出るかを
検証するためのバックテストシステム。**証券会社には接続しない検証専用**。

- 対象: 国内株式(現物)のデイトレード(同一日内のエントリー/エグジット)
- データ: [J-Quants API](https://jpx-jquants.com/)(JPX公式・無料プラン)
- 検証手法: `strategy.md` に整理(エントリー/エグジット/損切り/銘柄スクリーニング)

> ⚠️ これは過去データでの検証ツールです。将来の利益を保証するものではなく、
> 投資判断は自己責任で行ってください。

## ⚙️ 構成

```
src/
  data/          J-Quants API からのデータ取得・ローカル保存
  strategy/      strategy.md の手法をルールベースで実装(売買シグナル判定)
  backtest/      過去データに戦略を適用するシミュレーションエンジン
  report/        勝率・PF・最大DD などの指標算出とグラフ出力
dashboard/       Streamlit による確認用Webダッシュボード(PC/iPhone対応)
scripts/         データ取得などの実行スクリプト
data/            取得データの保存先(.gitignore 対象・コミットされない)
tests/           テスト
strategy.md      検証する手法の定義(※手法のnote共有後に作成)
```

## 🔑 セットアップ

```bash
# 1. 依存パッケージ
pip install -r requirements.txt

# 2. J-Quants の認証情報を設定
cp .env.example .env
#   .env を編集し、登録メールアドレス / パスワードを記入
#   (.env は .gitignore でコミット対象外)

# 3. 接続テスト
python scripts/fetch_data.py --check
```

### J-Quants 無料プランについて

- 取得可能期間: **直近約2年**
- **約12週間のデータ遅延**あり(直近営業日が数か月前になるのは正常)
- **分足(ザラ場)データは無料プランでは提供されない見込み**
  → 本システムはまず **日足ベースの検証**から始めます。
  日中の値動きを使う検証(寄り→引け、ザラ場の高値/安値タッチ等)は、
  日足の OHLC で近似できる範囲と、分足が必須で近似できない範囲を分けて扱います。

## 📊 使い方

```bash
# 1. 日足データ取得(要 J-Quants 登録)
python scripts/fetch_data.py --codes 7203 6758 9984 --from 2023-01-01 --to 2024-12-31

# 2. バックテスト(CLI)
python scripts/run_backtest.py --codes 7203 6758 9984
#   データ未取得でも合成データで動作確認:
python scripts/run_backtest.py --demo

# 3. ダッシュボード(ブラウザ・PC/iPhone対応)
streamlit run dashboard/app.py
#   スマホからは表示される Network URL にアクセス(同一Wi-Fi)
#   外出先からは --server.address 0.0.0.0 + トンネル等
```

### ダッシュボードでできること
- 資産推移グラフ(エクイティカーブ)+ ドローダウン
- 個別トレード一覧(日付・銘柄・損益・約定後資産)+ CSV出力
- サマリー指標(勝率・損益合計・PF・最大DD・期待値・シャープ)
- 対象銘柄・検証期間・戦略パラメータを画面で切り替え
- J-Quants 未登録でもデモデータで全機能を確認可能
