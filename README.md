# ピカピカスロット 🌟

ノーマルAタイプ（完全告知＋内部抽選方式）の挙動を再現したオリジナルスロットゲームのPWAです。
実機の公表確率・配当値に準拠しつつ、名称・デザイン・音はすべてオリジナルです。

## 遊び方（実機と同じ手順）

1. **MAX BET** でメダル3枚を投入（BETランプが点灯）
2. **レバー**を叩いてゲームスタート。内部抽選はこの瞬間に確定します
3. **STOP** ×3 で各リールを停止。小役は最大4コマすべりで自動的に引き込まれます
4. ボーナス当選時は **ピカランプ** が点灯（先告知1/4・後告知3/4、まれにレインボー＝プレミアム）
5. 点灯したらボーナス図柄（🌟🌟🌟=BIG／🌟🌟🟦=REG）を**目押しで**狙って揃えてください。ビタ押しでなくても4コマ手前までなら滑って止まります。揃うまで持ち越します

- リプレイ成立時は自動ベットされるので、そのままレバーを叩けばOK
- ゲーム間には実機同様の**4.1秒ウェイト**あり（WAITランプ点灯中はレバーを叩いてもウェイト明けにリールが始動）
- リール回転速度は実機準拠の**1周0.78秒**。回転はブラーなしのtransform駆動で目押し可能
- 有効ラインは3枚掛けの**5ライン**（上・中・下段＋斜め2本）。入賞ラインは点滅表示
- リール配列は実機（ゴーゴージャグラー系）の構造を再現：左リールは**チェリー付BAR**、
  右リールは**7の直下にBAR**。ぶどう・リプレイはどの押し順・押し位置でも取りこぼしなし
  （停止テーブルを事前計算して全押しパターン検証済み。`scripts/search-strips.mjs`参照）
- BIG＝純増約240枚、REG＝純増約96枚
- **AUTO** でフリー打ちの自動プレイ
- 筐体上部にホール設置型データカウンター風の表示（BIG/REG回数・回転数・総回転）
- クレジット・統計・実績・履歴はすべてlocalStorageに自動保存され、アプリを閉じても続きから遊べます

### 隠しメニュー

タイトルロゴを**5回タップ**すると「デバッグ」タブが出現します。
設定1〜6の変更、高速シミュレーション（最大10万G）、コイン補充、データリセットが可能です。

## 技術構成

- **Vite + React + TypeScript**（SPA）
- **PWA**: vite-plugin-pwa（Service Worker + manifest、オフライン起動対応）
- **状態管理**: Zustand（persistミドルウェアでlocalStorageに永続化）
- **スタイル**: CSS Modules（スマホ縦画面ファースト）
- **音**: Web Audio APIで効果音を自前合成（外部音源なし）
- **テスト**: Vitest（確率収束テスト）

ゲームのコアロジック（`src/game/`）はUIから完全に分離された純粋なTypeScriptモジュールで、
抽選・配当・リール停止制御はすべてテスト可能です。

```
src/
  game/        # 純粋なゲームロジック（抽選・配当・リール制御）
  state/       # Zustandストア（ゲーム状態・統計・実績）
  components/  # Reactコンポーネント
  audio/       # Web Audio効果音
tests/         # Vitest収束テスト
scripts/       # PWAアイコン生成
```

## 開発

```bash
npm install
npm run dev      # 開発サーバー起動
npm test         # 収束テスト実行（設定6で20万G等をシミュレート）
npm run build    # 型チェック + 本番ビルド（dist/）
npm run preview  # ビルド成果物の確認
```

PWAアイコンを再生成する場合: `node scripts/gen-icons.mjs`

### AI生成画像素材の差し替え

背景・マーキーパネル・ボーナス背景・リール図柄はAI生成画像を加工して使っています
（加工済みファイルは`src/assets/`にコミット済み）。素材を差し替える場合は、
`assets-src/`に以下を置いて`node scripts/prepare-assets.mjs`を実行してください。

- `bg.png` : 通常時の背景（縦長）
- `bonus.png` : ボーナス中の背景（縦長）
- `marquee.png` : ランプ後ろの装飾パネル（横長）
- `sheet.png` : 図柄シート（5列×3行。星/ブロック/ぶどう/チェリー/ベル + 2行目4列目にピエロ帽・5列目にリプレイ矢印）

スクリプトがリサイズ・WebP圧縮・図柄の切り抜き（白背景の透過化、
隣セルのはみ出し除去、正方形化）まで自動で行います。

## デプロイ

ビルド成果物は静的ファイルのみなので、任意の静的ホスティングにデプロイできます。

### Vercel

```bash
npm i -g vercel
vercel          # 初回はプロジェクト設定に従う（Framework: Vite）
vercel --prod
```

またはVercelのダッシュボードからGitHubリポジトリをインポートするだけでOKです
（Build Command: `npm run build` / Output Directory: `dist` が自動検出されます）。

### Netlify

Build command: `npm run build`、Publish directory: `dist` を指定。

### GitHub Pages

`vite.config.ts` の `base` は `'./'` にしてあるため、サブパス配下でもそのまま動作します。
`dist/` を `gh-pages` ブランチにpushするか、Actionsで自動デプロイしてください。

## iPhoneでのインストール

1. デプロイ先URLをSafariで開く
2. 共有ボタン → **ホーム画面に追加**
3. スタンドアロン（全画面）で起動し、オフラインでも遊べます
