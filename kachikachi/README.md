# カチカチくん 🔘🌺

パチスロ用の小役カウンター「勝ち勝ちくん」と同等の機能を持つ、単体で動作するPWAです。
ピカピカスロット本体とは独立したアプリで、単独でビルド・デプロイできます。
**ハナハナ（ハナハナホウオウ〜天翔〜）での使用を想定**しており、
示唆ランプの記録と設定推測機能を備えています。

## 機能（実機との対応）

| 実機の機能 | 本アプリでの操作 |
|---|---|
| 4色カウントボタン（白/赤/緑/黄） | カードをタップで +1（発光・カチッ音・バイブでフィードバック） |
| ▲押しながら減算（ミス修正） | カード内の専用 **−1** ボタン |
| ボタン長押しで個別リセット | カード内の **リセット** ボタン（誤操作防止の2段階確認） |
| 黒■▲ボタンで総回転数入力 | ±1/10/100/1000のステップボタン＋数値タップで直接入力 |
| 紫ボタンで÷（確率）モード切替 | カウントと確率（総回転数÷カウント）を**常時同時表示** |
| 紫→色ボタン2つで合算確率 | 合算パネルでチップを選択（**3役以上の合算にも対応**） |
| A/B/Cモード（3シーン独立） | シーンA/B/Cタブで切替（通常時・ボーナス中などを別カウント） |
| 電源OFFでもデータ保持 | localStorageに自動保存（閉じても復元） |
| NO SAVE電源ONで全消去 | 設定 → 全データをリセット |

アプリならではの追加機能:

- 役名ラベルの自由変更（カード左上の名前をタップ。デフォルトはハナハナ用: ベル/スイカ/チェリー）
- 画面スリープ防止（Wake Lock、設定でON/OFF）
- 音・バイブの個別ON/OFF
- オフライン完全動作（PWA、ホーム画面に追加可能）

## 設定推測タブ（ハナハナホウオウ〜天翔〜）

下部タブの「🌺 設定推測」で、ボーナスと示唆を記録して設定1〜6の期待度（ベイズ推定）を表示します。

**記録できるもの**

- BIG / REG 回数（総回転数はカウンタータブと共有）
- BIG中スイカ回数（確率はBIG回数×24G換算）
- REG中サイドランプ（スイカ成立時）: 青/緑=奇数示唆、黄/赤=偶数示唆、虹=高設定示唆
- BIG終了時トップランプ: 青＜黄＜緑＜赤＜虹の順に高設定に期待
- REG終了時トップランプ（確定演出）: 青=設定2以上/黄=3以上/緑=4以上/赤=5以上/虹=6確定

**推測に使う数値**

- ボーナス確率（公表値ベース）: BIG 1/297〜1/236、REG 1/496〜1/337
- ベル確率（実戦値目安）: 1/7.5〜1/7.2 — カウンタータブの「ベル」カウンターと連動
- BIG中スイカ（実戦値目安）: 約1/48〜1/33
- サイドランプ振り分け: 寒色:暖色 = 奇数6:4 / 偶数4:6 / 設定6は均等

確定演出は該当設定未満を除外し、その他はベイズ推定で事後確率を計算します。
ベル・スイカ・ランプの振り分けは解析サイトの実戦値に基づく**目安**であり、
推測結果は参考値です（数値は `src/logic/tensho.ts` で調整可能）。

長押しやモード切替は実機のボタン数の制約による操作系のため、
アプリでは専用ボタン・常時表示に置き換えて使いやすさを優先しています（機能は全て同等）。

## 開発

```bash
cd kachikachi
npm install
npm run dev      # 開発サーバー起動
npm test         # コアロジックの単体テスト
npm run build    # 型チェック + 本番ビルド（dist/）
npm run preview  # ビルド成果物の確認
```

PWAアイコンを再生成する場合: `npm run icons`

## 技術構成

- **Vite + React + TypeScript**（SPA / PWA）
- **状態管理**: Zustand（persistでlocalStorageに永続化）
- **音**: Web Audio APIでクリック音を自前合成（外部音源なし）
- **テスト**: Vitest（カウント・確率計算・合算・リセットの純関数テスト）

```
src/
  logic/counter.ts   # UI非依存の純関数ロジック（カウント・確率・合算）
  store.ts           # Zustandストア（シーンA/B/C・設定・永続化）
  sfx.ts             # クリック音の合成
  components/        # UIコンポーネント
tests/               # Vitestテスト
scripts/             # PWAアイコン生成
```

## デプロイ

ビルド成果物（`dist/`）は静的ファイルのみ。Vercel / Netlify / GitHub Pagesなど
任意の静的ホスティングにデプロイできます（Root Directory: `kachikachi`）。

## iPhoneでのインストール

1. デプロイ先URLをSafariで開く
2. 共有ボタン → **ホーム画面に追加**
3. スタンドアロン（全画面）で起動し、オフラインでも使えます
